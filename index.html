Methodology

Data Sources & Approach

  • Addresses: 309 unique Ohio addresses from the FBCJ ServantKeeper directory (3 geographic outliers excluded, 10 out-of-state excluded)
  • Geocoding: US Census Bureau geocoder (primary), Nominatim ZIP centroid fallback (9 addresses)
  • Driving data: Google Maps Distance Matrix API with Sunday 9:00 AM departure time for church-relevant traffic patterns
  • Contour maps: 40x40 grid (1,600 points) centered symmetrically on the midpoint between the two churches, covering a 20-mile radius in each direction. Smoothed with median filter + Gaussian blur to remove routing noise
  • Probability model: Hypothetical logistic model based on driving time difference. This is an assumed behavioral model, not fitted to actual attendance outcomes. It illustrates how attendance might shift based on convenience, but actual behavior depends on many factors (relationships, ministry preference, habit, etc.)

Important Notes

  • This dataset is inherently skewed toward Jackson — these are current FBCJ members, so most addresses cluster near the existing church
  • The contour maps show objective geographic data (which church is closer from any point). These are the most reliable visualizations
  • 9 addresses used ZIP code centroid (approximate location) and are marked as "Low Confidence" on the maps
  • All maps are interactive — hover over markers for family details, use layer controls to toggle views
